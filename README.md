# Game Mode Portfolio

게임 시작 화면처럼 진입하고, 프로젝트를 스테이지처럼 탐색하는 개인 포트폴리오 예시입니다.

## 최종 추천 방향

- **초기 버전:** 지금처럼 HTML/CSS/Vanilla JS 정적 사이트로 제작
- **업그레이드:** 프로젝트가 많아지면 Next.js + MDX로 전환
- **호스팅:** GitHub Pages로 먼저 배포하고, 포트폴리오를 본격 운영할 때 개인 도메인 연결

## 페이지 구성

- `Home`: 영상 배경, 로딩/Start 연출, CTA
- `Projects`: 프로젝트 카드, 태그, 상세 링크
- `About`: 소개와 스킬 상태창
- `Resume`: 이력 타임라인
- `Contact`: 메일, GitHub, LinkedIn

## 도메인 추천

- 취업/이직용이면 `이름.dev`, `이름.com`, `이름.me` 순서로 추천
- GitHub 중심이면 `username.github.io`로 시작해도 충분
- 장기적으로 브랜딩하려면 개인 도메인을 사서 GitHub Pages 또는 Vercel에 연결

## 실행

이 저장소는 정적 사이트라 `index.html`을 브라우저로 열면 바로 확인할 수 있습니다.
